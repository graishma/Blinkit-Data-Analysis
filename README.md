# BlinkIT Grocery Sales Analysis

## Project Overview
This project analyzes BlinkIT grocery sales data using SQL and Power BI to identify sales trends and business insights.

## Tools Used
- SQL (MySQL)
- Power BI
- Excel

## Dashboard Features
- Total Sales KPI
- Average Sales KPI
- Total Items KPI
- Average Rating KPI
- Sales by Item Type
- Sales by Outlet Type
- Sales by Outlet Size
- Sales by Outlet Location
- Interactive Slicers

## Business Insights
- Identified top-performing product categories.
- Compared outlet performance across locations and outlet types.
- Built an interactive dashboard for business decision-making.
